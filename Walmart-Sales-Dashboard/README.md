# 🛒 Walmart Sales Analytics Dashboard

## 📊 Project Overview

This project presents an end-to-end sales analysis of Walmart data using Power BI.
The dashboard is designed to provide insights into sales performance, customer behavior, and store operations to support data-driven decision making.

---

## 🧱 Data Model

The data is structured using a **star schema**:

* **Fact Table:** Sales_Fact
* **Dimension Tables:** Customer_Dim, Product_Dim, Store_Dim, Employee_Dim

This structure enables efficient analysis and scalable reporting.

---

## 🔍 Key Contributions

* Integrated multiple datasets and established relationships
* Performed data cleaning and transformation using Power Query
* Created calculated measures using DAX
* Designed an interactive dashboard for business insights

---

## 📈 Key Insights

* Sales and profit trends over time
* Store-wise performance comparison
* Product category contribution to revenue
* Customer segmentation patterns

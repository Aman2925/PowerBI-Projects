# 📦 Supply Chain Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing supply chain operations to identify delays, inefficiencies, and operational performance using Power BI.

The dataset was initially messy and unstructured. It was cleaned and transformed using Python before being used for dashboard development.

---

## ⚙️ Tech Stack
- Python (Pandas, NumPy)
- Power BI
- Excel / CSV

---

## 🧹 Data Cleaning Process
- Handled missing values
- Standardized inconsistent category names
- Converted date columns into proper format
- Removed duplicates
- Created new calculated fields

All cleaning steps are available in:
📁 `notebooks/data_cleaning.ipynb`

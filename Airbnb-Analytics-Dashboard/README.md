# 🏡 Airbnb Analytics Dashboard (Power BI)

## 📊 Project Overview

This project analyzes Airbnb listing data to understand pricing trends, availability patterns, and customer preferences.
The dashboard helps identify key factors that impact listing performance and supports better decision-making.

---

## 🔍 Key Analysis Areas

* Price distribution across locations
* Room type performance
* Availability trends
* Review and rating insights

---

## 🧠 What I Did

* Cleaned and transformed data using Power Query
* Built calculated measures using DAX
* Designed an interactive dashboard for analysis

---

## 📈 Key Insights

* Pricing varies significantly by location and room type
* Entire homes tend to generate higher revenue
* Listings with better ratings show higher demand
* Availability patterns highlight seasonal demand trends

---

## 🛠 Tools Used

* Power BI
* Power Query
* DAX

---

## 📂 Dataset

Dataset sourced from Maven Analytics Airbnb dataset:
🔗 https://mavenanalytics.io/data-playground/airbnb-listings-reviews
